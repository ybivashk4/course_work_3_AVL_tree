Для запуска программы необходимо открыть файл avl_tree.dev с помощью IDE dev cpp и скомпилировать программу. 
Если же данный IDE отсутствует, необходимо создать проект в IDE поддерживающей компиляцию c++ и добавить в него файлы func.cpp, main.cpp, menu.cpp, tree.h

Далее появится файл avl_tree.exe - запускаем его и следуем инструкциям на экране.

Если необходимо можно записать свой набор значений в папке saves. Заходим в эту папку, создаём там файл любого типа, открываем с помощью текстового редактора и через пробел записываем значения. 

Также есть программа, сохраняющая в файл с именем test.txt 1e7 значений начиная с 1 и до 1e7, чтобы запустить её необходимо зайти в папку test, открыть файл test.cpp и скомпилировать его. Запускаем test.exe, после чего заходим в avl_tree.exe - вставляем слудующие:
2 1 large.txt 
и ждём до 5 минут (у меня загрузилось за 30 секунд) пока информация будет загружаться.
